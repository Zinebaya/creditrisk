# AstraRisk Frontend

Enterprise-grade fintech SaaS frontend built with Next.js App Router, React, TailwindCSS, Framer Motion, and Recharts.

## Features

- Landing page with hero, pricing, contact, and stats
- Login page with authentication UI
- Admin dashboard with charts, analytics, and activity cards
- Manual credit risk prediction page with AI-ready result panels
- CSV/XLS upload page with drag-and-drop and results table
- Client history table with filters and export actions
- Admin management page with role-based UI
- Profile edit page with security controls
- Subscription and billing page
- Analytics page with line, bar, and pie charts
- Reusable component library and premium design system

## Setup

```bash
cd frontend
npm install
npm run dev
```

The app runs at `http://localhost:3000` by default.

## Project structure

- `/app` — Next.js page routes
- `/components` — shared design system and reusable UI components
- `/lib` — utility helpers
- `/types` — shared TypeScript types
